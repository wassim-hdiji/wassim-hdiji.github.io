# wassim-hdiji.github.io

This repository contains a simple static portfolio site scaffold.

Getting started

1. Open `index.html` in a browser to view the site.
2. Edit `index.html`, `css/style.css`, and `js/main.js` to customize the content.

Project structure

- `index.html` — main page
- `css/style.css` — stylesheet
- `js/main.js` — client-side script
